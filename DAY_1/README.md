## Key HTML Concepts Covered

- **Boilerplate** — `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`, `<meta>`
- **Text Tags** — `<h1>`–`<h6>`, `<p>`, `<strong>`, `<i>`, `<em>`, `<b>`,
  `<br>`, `<hr>`
- **Media** — `<img>`, `<video>`, `<iframe>`
- **Links** — `<a href="">`, navigation between pages
- **Forms** — `<form>`, `<input>`, `<label>`, `<select>`, `<option>`,
  `<button>`, `required`, `placeholder`
- **Lists** — `<ul>`, `<ol>`, `<li>`, nested lists
- **Tables** — `<table>`, `<tr>`, `<th>`, `<td>`
- **Semantic Tags** — `<header>`, `<nav>`, `<section>`, `<article>`, `<aside>`,
  `<main>`, `<footer>`
- **Styling** — inline styles (`style=""`), external CSS via `<link>`
